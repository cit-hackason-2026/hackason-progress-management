# 調達計画

## 調達対象
外部から調達するサービス・ライブラリ・データ・物品を管理する。

| 項目 | 種別 | 提供元 | ライセンス/プラン | 費用 | 利用期間 | 担当 |
|------|------|--------|------------------|------|---------|------|
| | クラウド / API / ライブラリ / データセット / 物品 | | | | | |

## ライセンス確認
- [ ] OSSライセンスの利用条件（MIT / Apache-2.0 / GPL など）
- [ ] データセットの再配布・商用利用可否
- [ ] 画像・フォント・音源の利用規約

## 承認フロー
1. 担当者が候補を提案
2. ライセンス・費用・代替案をリーダーが確認
3. 承認後に契約/アカウント作成
4. 本ファイルに追記
